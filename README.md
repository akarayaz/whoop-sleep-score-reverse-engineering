# Reverse-Engineering WHOOP's Sleep Performance Score

A single-subject (n=91 nights) case study reconstructing WHOOP's official Sleep Performance
formula from its 4 documented components, using linear regression validated with
leave-one-out cross-validation (out-of-sample R² = 0.922).

Includes a deeper investigation into why one component (Sleep Efficiency) initially showed
no effect, using a threshold test and a statistical power analysis.

**Author:** Alper Karayaz — Data Science & Engineering, Politecnico di Torino
